# ValidateSubsequence

Given two non-empty arrays of integers, write a function that determines whether the second array is a subsequence of the first one.

A subsequence of an array is a set of numbers that aren’t necessarily adjacent in the array but that are in the same order as they appear in the array. For example these numbers [2, 3, 5] are a subsequence of the array: [1, 2, 3, 4, 5]
