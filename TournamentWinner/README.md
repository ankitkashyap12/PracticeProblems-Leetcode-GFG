# TournamentWinner

There's an algorithms tournament taking place in which teams of programmers compete against each other. Each programming team has a certain number of victories. A team wins 3 points for each victory and 0 points for a loss. The tournament is represented by a list of pairs, each pair containing the name of the winning team and the name of the losing team. No draws occur.

Write a function that returns the name of the team with the most points. The winning team of each pair gets 3 points and the losing team gets 0 points. If two teams have the same number of points, return the name of the team that appears first in the list of pairs.
